# FDA Adverse Event Reports Analysis (2015-2026)

An exploratory data analysis of drug adverse event reports submitted to the FDA's FAERS (Adverse Event Reporting System) database between 2015 and 2026.

## Overview

This project examines 528,000 adverse event reports across 30 features, focusing on the most frequently reported drugs, fatal vs. non-fatal outcomes, reporting trends over time, and a deep dive into reactions associated with Sodium Oxybate (Xyrem).

## Research Questions

- Which drugs have the highest number of adverse event reports?
- What proportion of reports result in fatal outcomes?
- How have reporting trends changed over time?
- What are the most common adverse reactions for Sodium Oxybate?

## Data Source

FDA Adverse Events 2015-2026 (CLEAN) via Kaggle — 528,000 reports, 30 features

## Tools & Libraries

Python, pandas, matplotlib, seaborn, Jupyter Notebook

## Visualizations

- Bar chart: Top 10 most reported drugs
- Bar chart: Fatal vs non-fatal outcomes by drug
- Line chart: Reporting trends over time with Sodium Oxybate highlighted
- Bar chart: Top 10 adverse reactions for Sodium Oxybate
